How to use this API

  • Health check: /health
  • Available conflicts: /conflicts
  • Ukraine feed: /scrape?conflict=ukraine&max_incidents=100
  • Taiwan feed: /scrape?conflict=taiwan&max_incidents=100
  • Custom URL mode: /scrape?url=https://iran.liveuamap.com/&max_incidents=100
  • Use only one of conflict or url.
  • Data is from LiveUAMap source pages.
